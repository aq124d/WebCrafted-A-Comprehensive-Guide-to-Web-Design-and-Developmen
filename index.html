Reference

The concept of web design and development has been around for about as long as websites have existed. The process of developing a website is referred to as web design and development. It involves two main skill sets, as the name implies: web development and web design. A website's functionality is determined by web development, but its appearance and feel are determined by web design [12].

Web-Design

Web design refers to the visual design and experiential aspects of a particular website. We're going to dive into more detail about web design throughout the rest of this article [9].

Web-Development

Web Development refers to the creation and upkeep of a website's framework, which entails complex coding techniques to guarantee optimal website functionality[6]. There are two categories under web-development

  • Front-end development
  • Back-end development

Front-end development

A front-end developer is a kind of software engineer that focuses on developing and designing websites' and web applications' user interfaces (UI) and user experiences (UX). A front-end developer's main duty is to make sure that an application or website's visual and interactive elements are intuitive, visually beautiful, and functionally sound [5].

Front-end developers work with various technologies, tools, and languages, including:

  • HTML (HyperText Markup Language): The standard markup language used to create the structure and layout of web pages.
  • CSS (Cascading Style Sheets): A stylesheet language used to control the presentation, formatting, and appearance of web pages.
  • JavaScript: A programming language that allows developers to add interactivity, animations, and other dynamic elements to websites and web applications.

Back-end development

Working on server-side software, which focuses on everything you can't see on a website, is known as back-end development. With an emphasis on databases, back-end logic, application programming interfaces (APIs), architecture, and servers, back-end developers make sure the website operates as intended. They make use of code that facilitates browser-DB communication as well as data storage, comprehension, and deletion [6].

    Programming Languages

  • JavaScript (Node.js): Popular for building scalable and fast server-side applications. Node.js allows developers to use JavaScript for both frontend and backend development.
  • Python: Known for its readability and versatility, Python is often used with frameworks like Django and Flask for backend development.
  • Ruby: Used in conjunction with the Ruby on Rails framework for rapid and efficient web application development.

    • Databases

  • Relational Databases: Such as MySQL, PostgreSQL, and SQLite.
  • NoSQL Databases: Such as MongoDB, Cassandra, and Redis for handling unstructured or semi-structured data.

    • Containerization and Orchestration

  • Docker: A platform for developing, shipping, and running applications in containers.
  • Kubernetes: An open-source container orchestration platform for automating the deployment, scaling, and management of containerized applications.
